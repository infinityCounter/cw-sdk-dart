OrderBookSnapshot class

An OrderBookSnapshot is the deterministic state of an OrderBook at any given point in time.

An OrderBookSnapshot is immutable.

OrderBookSnapshot(Iterable<PublicOrder> asks, [ Iterable<PublicOrder> bids, [ int seqNum = 0 ])
Instantiates a new OrderBookSnapshot object using an Iterable set of asks and bids PublicOrders. Optionally a seqNum can be passed.
